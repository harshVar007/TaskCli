# TaskCli
https://github.com/harshVar007/TaskCli
### TaskCli in Java - In Progress

### Usage 
``Usage: add <value> | delete <id> | setStatus <id> <status> | filter <status> | help | getAll``

#### Options
1. add \<value> : add value to the list
2. delete \<id> : delete task at Id
3. setStatus \<id> \<status> : setStatus at task id
4. filter \<status> : filter tasks by Status
5. help : display help message
6. getAll : return all Tasks

### How to Use
#### Prerequisite - Java Insalled and on Path
##### Example
``java -jar TaskCli.jar [add|delete|getAll] <value|id|status>``
